# 减脂日记

一个专为减脂期设计的 PWA（渐进式网页应用），用于记录每日饮食和运动打卡。基于个人减脂食谱、运动计划和动作库数据构建，支持离线使用，数据保存在本地浏览器中。

---

## 功能特性

### 首页 — 每日打卡仪表盘
- **日期导航**：前后切换日期，自动匹配对应星期食谱
- **饮食打卡**：早餐、午餐、加餐、晚餐 4 餐分别勾选
- **运动打卡**：根据当前运动周显示当天计划，一键标记完成
- **饮水记录**：7 个时段打卡 + 实时进度条
- **统计概览**：3 个数据卡片实时展示当天完成度

### 饮食页 — 四大模块
- **本周食谱**：周一到周日完整食谱，每餐可展开查看详细食材
- **饮食原则**：6 大清洁饮食原则 + 避免/推荐食材清单
- **饮水计划**：2000ml 分 7 个时段的饮水安排
- **烹饪建议**：推荐烹饪方式优先级 + 调味料选择指南

### 运动页 — 计划与动作库
- **运动计划**：4 周进阶计划（适应期 → 巩固期 → 提升期 → 稳定期），支持周切换
- **动作库**：
  - 力量训练（上肢 / 下肢 / 核心）
  - 热身动作
  - 拉伸动作

### 参考页 — 信息总览
- **日历记录**：
  - 折叠态：当前周一到周日预览，今天高亮
  - 展开态：完整月历，类 GitHub 贡献图的打卡热力图（4 级色块）
  - 统计：连续打卡天数、累计打卡天数、本月打卡天数
- **个人信息**：身高、体重、BMI、热量目标、营养素比例
- **运动安全提示**：关节保护、运动前后注意事项
- **外出就餐指南**：餐厅选择优先级 + 点餐原则
- **预期效果**：分阶段减重预期
- **关键里程碑**：从 190 斤到 140 斤的减重节点

### 打卡程度逻辑

| 等级 | 条件 | 日历色块 |
|------|------|---------|
| Level 0 | 无模块完成 | 透明 |
| Level 1 | 完成 1 个模块 | 浅绿 |
| Level 2 | 完成 2 个模块 | 中绿 |
| Level 3 | 全部完成 | 深绿 |

- **饮食完成**：早餐 + 午餐 + 晚餐全部勾选（加餐不计入）
- **运动完成**：点击"完成运动"按钮
- **饮水完成**：只要饮食或运动完成任意一个，饮水自动视为完成

## 项目结构

```
fitness-pwa/
├── index.html          # 主页面（4 个 tab：首页、饮食、运动、参考）
├── style.css           # 全局样式，深色主题 + emerald green 主色调
├── app.js              # 核心交互逻辑（打卡、日历、页面切换）
├── data.js             # 饮食/运动数据（食谱、计划、动作库）
├── manifest.json       # PWA 配置
├── sw.js               # Service Worker 离线缓存
├── icons/
│   └── icon.svg        # 应用图标
└── README.md           # 本文档
```

---

## 技术栈

- **纯前端**：HTML5 + CSS3 + Vanilla JavaScript
- **PWA**：Service Worker + Web App Manifest
- **数据持久化**：localStorage
- **离线支持**：核心资源预缓存，离线可正常使用
- **深色主题**：适配 OLED 屏幕，省电护眼

## 使用方式

### 本地预览

```bash
# 进入项目目录
cd fitness-pwa

# 启动本地服务器
python -m http.server 8080

# 浏览器打开 http://localhost:8080
```

### 部署到 iPhone 使用（推荐）

1. 将项目上传到 **GitHub Pages**（免费、自动 HTTPS）
2. 在 iPhone Safari 中打开部署地址
3. 点击底部"分享"按钮 → "添加到主屏幕"
4. 桌面会出现"减脂日记"图标，点击即可像原生 App 一样使用

> 注意：iPhone 上 PWA 的推送通知和部分高级功能有系统限制，但核心打卡和数据保存功能完全可用。

## 数据来源

项目中的饮食食谱、运动计划和动作库均来自用户提供的个人减脂方案文档：

- `减脂饮食食谱.md` — 一周食谱、饮水计划、烹饪建议
- `减脂运动清单.md` — 四周进阶运动计划
- `力量训练与热身拉伸动作库.md` — 详细动作说明

## 自定义修改

### 修改个人信息
编辑 `index.html` 中参考页面的个人信息区块。

### 修改食谱
编辑 `data.js` 中的 `WEEKLY_DIET` 对象。

### 修改运动计划
编辑 `data.js` 中的 `WORKOUT_PLANS` 对象。

### 修改动作库
编辑 `data.js` 中的 `EXERCISE_LIBRARY`、`WARMUP_EXERCISES`、`STRETCH_EXERCISES`。
